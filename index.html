Welcome

Browse the Gray Whale API reference documentation. These Swagger pages are interactive and give you access to all of the endpoints needed to build and deploy your self-learning recommendation feed using your custom Large Interaction Model (LIMs).

If you're looking for more documentation for the Gray Whale Platform, or would prefer to interact with our API using our custom SDK, please visit the Remoras Documentation.
